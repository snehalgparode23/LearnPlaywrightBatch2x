# Visual Studio Code - Windows Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + P` | Open Command Palette |
| `Ctrl + P` | Quick Open (Go to File) |
| `Ctrl + Shift + N` | New Window |
| `Ctrl + W` | Close Window / Tab |
| `Ctrl + K, Ctrl + S` | Open Keyboard Shortcuts |
| `Ctrl + ,` | Open Settings |
| `Ctrl + B` | Toggle Sidebar |
| `Ctrl + \`` | Toggle Integrated Terminal |
| `Ctrl + Shift + Q` | Quit VS Code |
| `Ctrl + N` | New File |
| `Ctrl + O` | Open File |
| `Ctrl + S` | Save |
| `Ctrl + Shift + S` | Save As |
| `Ctrl + K, Ctrl + W` | Close All Editors |
| `Ctrl + Shift + T` | Reopen Closed Editor |
| `Ctrl + Tab` | Open Next Editor |
| `Ctrl + Shift + Tab` | Open Previous Editor |
| `Ctrl + PageDown` | Open Next Editor |
| `Ctrl + PageUp` | Open Previous Editor |
| `Ctrl + K, Ctrl + O` | Open Folder |
| `Ctrl + K, Ctrl + F` | Close Folder |
