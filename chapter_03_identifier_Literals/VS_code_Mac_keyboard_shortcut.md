# Visual Studio Code - Mac Keyboard Shortcuts

## General

| Shortcut           | Action                          |
| ------------------ | ------------------------------- |
| `Cmd + Shift + P`  | Open Command Palette            |
| `Cmd + P`          | Quick Open (Go to File)         |
| `Cmd + Shift + N`  | New Window                      |
| `Cmd + W`          | Close Window / Tab              |
| `Cmd + K, Cmd + S` | Open Keyboard Shortcuts         |
| `Cmd + ,`          | Open Settings                   |
| `Cmd + B`          | Toggle Sidebar                  |
| `Ctrl + \``        | Toggle Integrated Terminal      |
| `Cmd + Q`          | Quit VS Code                    |
| `Cmd + N`          | New File                        |
| `Cmd + O`          | Open File                       |
| `Cmd + S`          | Save                            |
| `Cmd + Shift + S`  | Save As                         |
| `Cmd + K, Cmd + W` | Close All Editors               |
| `Cmd + Shift + T`  | Reopen Closed Editor            |
| `Cmd + Tab`        | Open Next Editor (Mac OS level) |
| `Cmd + PageDown`   | Open Next Editor                |
| `Cmd + PageUp`     | Open Previous Editor            |
| `Cmd + K, Cmd + O` | Open Folder                     |
| `Cmd + K, Cmd + F` | Close Folder                    |

## Basic Editing

| Shortcut              | Action                      |
| --------------------- | --------------------------- |
| `Cmd + X`             | Cut line (empty selection)  |
| `Cmd + C`             | Copy line (empty selection) |
| `Cmd + Shift + K`     | Delete line                 |
| `Cmd + Enter`         | Insert line below           |
| `Cmd + Shift + Enter` | Insert line above           |
| `Cmd + Shift + \`     | Jump to matching bracket    |
| `Cmd + ]`             | Indent line                 |
| `Cmd + [`             | Outdent line                |
| `Home`                | Go to beginning of line     |
| `End`                 | Go to end of line           |
| `Cmd + Up`            | Go to beginning of file     |
| `Cmd + Down`          | Go to end of file           |
| `Ctrl + Up`           | Scroll line up              |
| `Ctrl + Down`         | Scroll line down            |
| `Alt + PageUp`        | Scroll page up              |
| `Alt + PageDown`      | Scroll page down            |
| `Cmd + Shift + [`     | Fold region                 |
| `Cmd + Shift + ]`     | Unfold region               |
| `Cmd + K, Cmd + [`    | Fold all subregions         |
| `Cmd + K, Cmd + ]`    | Unfold all subregions       |
| `Cmd + K, Cmd + 0`    | Fold all regions            |
| `Cmd + K, Cmd + J`    | Unfold all regions          |
| `Cmd + K, Cmd + C`    | Add line comment            |
| `Cmd + K, Cmd + U`    | Remove line comment         |
| `Cmd + /`             | Toggle line comment         |
| `Shift + Option + A`  | Toggle block comment        |
| `Cmd + Z`             | Undo                        |
| `Cmd + Shift + Z`     | Redo                        |
| `Cmd + K, Cmd + Z`    | Undo cursor position        |
| `Cmd + U`             | Soft undo                   |
| `Cmd + Shift + U`     | Show Output panel           |

## Multi-cursor and Selection

| Shortcut                             | Action                                      |
| ------------------------------------ | ------------------------------------------- |
| `Option + Click`                     | Insert cursor                               |
| `Cmd + Option + Up`                  | Add cursor above                            |
| `Cmd + Option + Down`                | Add cursor below                            |
| `Cmd + U`                            | Undo last cursor operation                  |
| `Shift + Option + I`                 | Insert cursor at end of each line selected  |
| `Cmd + L`                            | Select current line                         |
| `Cmd + Shift + L`                    | Select all occurrences of current selection |
| `Cmd + F2`                           | Select all occurrences of current word      |
| `Shift + Option + Right`             | Expand selection                            |
| `Shift + Option + Left`              | Shrink selection                            |
| `Shift + Option + (drag mouse)`      | Column (box) selection                      |
| `Cmd + Shift + Option + (arrow key)` | Column (box) selection                      |
| `Cmd + Shift + Option + PageDown`    | Column (box) selection page down            |
| `Cmd + Shift + Option + PageUp`      | Column (box) selection page up              |
| `Cmd + D`                            | Add selection to next find match            |
| `Cmd + K, Cmd + D`                   | Move last selection to next find match      |
